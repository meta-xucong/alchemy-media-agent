# 07. 数据模型与状态机

## 命名规则

2.0 数据必须独立：

1. 表名前缀：`v2_`。
2. Redis 前缀：`alchemy:v2:`。
3. 对象存储 prefix：`v2/`。
4. Trace project：`alchemy-media-agent-v2`。

## CreativeRun 状态机

```text
created
  -> planning
  -> retrieving_cases
  -> composing_prompt
  -> safety_checking
  -> generating
  -> reviewing
  -> completed
```

失败状态：

```text
failed
cancelled
blocked_by_policy
waiting_for_user
```

## SyncRun 状态机

```text
queued
  -> fetching
  -> parsing
  -> classifying
  -> indexing
  -> validating
  -> publishing
  -> completed
```

失败状态：

```text
failed
partially_completed
skipped_no_change
```

## 数据保留

1. 原始 provider snapshot 可保留最近 5 版。
2. 生图输出按用户套餐和合规策略保留。
3. trace 中的敏感内容需要脱敏。
4. 用户上传素材必须支持删除。
5. 案例库 provenance 永久保留。
