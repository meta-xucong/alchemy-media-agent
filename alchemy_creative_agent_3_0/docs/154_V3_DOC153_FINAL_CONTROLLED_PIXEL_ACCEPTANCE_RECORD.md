## 4. Final decision and limits

Doc153 is accepted for this shared Human Realism milestone. The current result
is suitable to close this refinement round: the system can produce natural
closed-mouth, quiet, and situation-grounded expressions instead of forcing one
standard commercial smile.

This acceptance does not claim that every generic “gentle natural smile” will
avoid all commercial-smile resemblance, nor does it claim certified Web
Provider delivery, Gate C/D passage, or production readiness. Any later
improvement must remain in the shared Brain/Human Realism/review path and must
not become a child-specific prompt recipe, keyword list, regular-expression
classifier, local prompt patch, or private retry path.
