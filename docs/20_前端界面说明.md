# 20. 前端界面说明

## 1. 当前状态

已新增挂载在 FastAPI 上的桌面前端与移动 H5 前端：

```text
src_skeleton/app/static/index.html
src_skeleton/app/static/styles.css
src_skeleton/app/static/app.js

src_skeleton/app/mobile_static/index.html
src_skeleton/app/mobile_static/mobile.css
src_skeleton/app/mobile_static/mobile.js
```

访问地址：

```text
http://127.0.0.1:8017/
http://127.0.0.1:8017/h5
http://127.0.0.1:8017/mobile
```

部署到正式域名后，建议默认使用同一域名内分页：

```text
https://your-domain.example/
https://your-domain.example/h5
https://your-domain.example/mobile
```

桌面版顶部提供“手机 H5”入口；移动 H5 顶部提供“桌面版”入口。这样本地、GitHub 和 VPS 部署路径保持一致。

## 2. 设计语言

桌面版采用轻奢工作台风格；移动 H5 采用 Gallery Ritual 视觉语言。

桌面版：

- 清晰的信息层级。
- 系统字体栈。
- 克制的中性色底、绿色主操作、少量状态色。
- 半透明材料感与轻量阴影。
- 分段控件、滑杆、文件选择、状态胶囊和紧凑面板。

移动 H5：

- 近白画廊底色。
- 高字距 serif 品牌锁定。
- 下划线式模块切换。
- 低饱和鼠尾草绿色主操作。
- 细线、轻边框、少阴影。
- 默认只露出主流程，其余能力统一收进每个模式的“高级”入口；历史图片保持独立展示。

不使用 Apple 商标、Logo、专有图标或专有素材。

## 3. 已接入接口

| UI 区域 | 接口 |
|---|---|
| 会话 | `POST /v1/sessions` |
| Provider 列表 | `GET /v1/providers` |
| 素材 | `POST /v1/assets/upload-url`、`POST /v1/assets/{asset_id}/complete` |
| 生图 | `POST /v1/image/jobs`，默认调用真实图片 provider |
| 下载 | `GET /v1/outputs/{output_id}/download` |
| 炼金 | `POST /v1/image/jobs/{job_id}/revise` |
| 视频预留 | `POST /v1/video/jobs` |
| 事件 | `GET /v1/sessions/{session_id}/events` |
| V2 案例与 Agent | `/api/v2/*`，本地 8017 会指向 `http://127.0.0.1:8020/api/v2` |

## 4. 验证结果

- 后端测试：`16 passed`

- 移动视口：390px 宽无横向溢出。
- 控制台：无 JS error。
