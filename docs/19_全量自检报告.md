# 19. 全量自检报告

自检时间：2026-06-03

## 1. 自检结论

当前代码骨架在“后端 MVP / mock 环境 / 生图链路先完整落地 / 视频接口预留”的范围内可运行、可测试、无已知可复现逻辑 BUG。

本次自检发现并修复了以下问题：

- OpenAPI 中的素材端点和 SSE 端点未挂到 FastAPI。
- 未知 image/video provider 可能走到未捕获异常。
- 炼金接口未校验 `output_id` 是否属于传入的 `job_id`。
- mock provider 对 `jpeg` / `webp` 输出的文件字节与声明格式不一致。
- HTTP 冒烟未覆盖下载、素材、SSE 和 unknown provider。

## 5. 验收建议

以当前代码骨架验收时，建议按以下口径：

- 验收通过：后端 mock MVP 生图链路、provider contract、素材/SSE/API 协议、视频预留接口。
- 条件通过：真实供应商调用，需要配置 API Key 并切换 `MEDIA_AGENT_MODE=live` 后另做 live 验证。
- 不纳入本次通过标准：前端、生产 DB/队列/对象存储、真实素材解析和真实视频生成。
