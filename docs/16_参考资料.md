# 16. 参考资料

资料基线：2026-06-02。以下链接建议在进入开发前再次确认版本、价格、限流和区域可用性。

1. OpenAI Agents SDK - Agents
   - URL: https://openai.github.io/openai-agents-python/agents/
   - 备注：Agent、Runner、tools、handoffs、guardrails、sessions 等核心概念。

2. OpenAI Agents SDK - Running agents
   - URL: https://openai.github.io/openai-agents-python/running_agents/
   - 备注：Runner 的 agent loop：模型输出、tool calls、handoff、final_output、streaming。

3. OpenAI Agents SDK - Models
   - URL: https://openai.github.io/openai-agents-python/models/
   - 备注：OpenAI 默认模型、非 OpenAI provider、ModelProvider、Agent.model、LiteLLM/Any-LLM 适配方式。

4. OpenAI API - Image generation

   - URL: https://developers.openai.com/api/docs/models/gpt-image-2
